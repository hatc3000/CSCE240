The area and perimeter variables are set in the Shape class.  The values to populate these variables are calculated in the constructors of the child classes.

To run the program, go to the src documentary and run the main.exe file.

For the triangle operations if two variables are entered it will only be able to calculate the area.  Three variables will enable it to calculate perimeter.

For rectangle operations, it will take the first two inputted variables and calculate both perimeter and area.

The circle operates both area and perimeter based off the integer in the second line of data/input.txt
